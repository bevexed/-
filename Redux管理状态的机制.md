#### Redux

1. 独立的状态管理 JS 库
2. 可以结合 react，angular，vue等使用
3. 集中式管理 react 应用中多个组件共享的状态，和后台获取数据



##### 扩展

- react-redux 简化 redux 编码
- redux-thunk 实现 redux 异步编程
- redux devtools 实现 chrome 中 redux 的调试





##### 机制

- store
  - 内部包含 state
  - 方法：
    - getState()

- reducers

- action creator
