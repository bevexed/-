#### 执行上下文（execute context）

1. 理解：代码执行的环境
2. 时机：代码正式执行之前会进入到执行环境

3. 工作：

   1. 创建变量对象

      1. 变量
      2. 函数及函数的参数
      3. 全局：window
      4. 局部：抽象的但是确实纯在

   2. 确认 this 的指向

      1. 全局：this ---> window
      2. 局部：this ---> 调用其的对象

   3. 创建作用域链

      1. 父级作用域链 + 当前的变量对象

   4. 扩展

      1. ECObj = {

          		变量对象：{变量，函数，函数的形参}，

         ​		scopeChain：父级作用域链 + 当前的变量对象，

         ​		this：{window || 调用其的对象}

         ​	}
