# ES6新特性

## let

1. 不存在变量提升
2. 同一个作用域下不能重复定义同一个名字
3. 有着严格的作用域 （var 是函数作用域）

## 箭头函数
1. 箭头函数的 this 在定义时就决定了（this 指向定义时外层定义的第一个普通函数的this），而且不可修改(不受 call、bind、apply 影响)
2. 不能 new（不能作为构造函数）
3. 不存在 prototype
4. 不存在 arguments

## Set

1. 数组去重
2. 交集，差集，并集
